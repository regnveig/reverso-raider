# Reverso Raider

Free tool to get Reverso Context examples uncensored.

Author: regnveig

## Dependencies

* Python 3.x
* Python packages: argparse, pandas, termcolor

## Syntax

One query mode:

```bash
python3 raider.py -s english -t russian -p "anything translatable"
```
Multi-query mode:

```bash
python3 raider.py -s english -t spanish
```
